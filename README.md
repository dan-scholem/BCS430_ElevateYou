# BCS430_ElevateYou
ElevateYou is a one-stop-shop health application which delivers all health-related features to users in an intuitive package, touching on all aspects of
health including physical and mental.


Technologies:
 - Java
 - JavaFX
 - IDE: IntelliJ
 - Frontend: SceneBuilder and CSS
 - Database: Firebase Authentication & Firestore Database
 - APIs: RapidAPI, OpenAI, NewsAPI, NPPES API

Feature List:

1) User Authentication and Registration:
   - Users can create an account and login using email and password.


2) Interactive Calendar
   - Users can add any upcoming events or reminders and view them.


3) Medication Tracker
   - Users can enter any medications they are taking.
   

4) Personal Journal
   - Users can create personal journal entries and enter their current mood.
   
   
5) Notifications
   - Users can receive notifications for medication, appointment, event, and hydration reminders.
   

6) Calories and Water Intake Tracker
   - Users can add a food and its amount of calories.
   - Users can enter the amount of water they drank in ounces.
   - Calorie goals and weight can also be entered.
   

7) Appointment Scheduling
   - Users can schedule their doctor's appointments.
   - Users can search for doctors in their area.
   

8) Sleep Tracking
    - Users can track their sleep and receive a sleep score.
   

9) Profile Settings
   - Users can update their profile settings including their email, password and user bio.
   

10) Exercise Tracking
    - Users can choose a specific workout and track their workout progress.
    

11) Articles
    - Users can search for and receive articles and save their favorites.
    

12) Friends
    - Users can search for people on the app and send a friend request.
    - Users can and view another user's profile or block them.
    

13) Quotes and Affirmations
    - Users can obtain quotes or affirmations and create their own personal affirmation.
    

14) Live Chat
    - Users can chat with a live chatbot and gain health insights.
    

15) Data Exports and Privacy
    - Users can export their health data into an emergency card.
    

16) Dashboard Insights
    - Users can view insights on their weight, calories, sleep score, and medication.
